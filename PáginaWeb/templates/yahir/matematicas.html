Puntos Básicos de Matemáticas

1. Operaciones Básicas

  • Suma: \(a + b = c\)
  • Resta: \(a - b = c\)
  • Multiplicación: \(a \times b = c\)
  • División: \(a \div b = c\)

Ejemplo: Si \(a = 5\) y \(b = 3\):

Suma: \(5 + 3 = 8\)

Resta: \(5 - 3 = 2\)

Multiplicación: \(5 \times 3 = 15\)

División: \(5 \div 3 \approx 1.67\)

2. Fórmulas Clásicas

Algunas fórmulas importantes incluyen:

  • Área de un círculo: \(A = \pi r^2\)
  • Teorema de Pitágoras: \(a^2 + b^2 = c^2\)
  • Ecuación cuadrática: \(ax^2 + bx + c = 0\)

Ejemplo: Encuentra el área de un círculo con radio \(r = 4\):

\(A = \pi (4)^2 = 16\pi \approx 50.27\)

3. Funciones

Algunas funciones comunes en matemáticas son:

  • Lineales: \(f(x) = mx + b\)
  • Cuadráticas: \(f(x) = ax^2 + bx + c\)
  • Exponenciales: \(f(x) = a \cdot e^{bx}\)

Ejemplo: Para una función lineal con \(m = 2\) y \(b = 3\):

\(f(x) = 2x + 3\)

Si \(x = 5\), entonces \(f(5) = 2(5) + 3 = 13\).
